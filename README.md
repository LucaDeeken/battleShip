# Battleship

A classic Battleship game built in **JavaScript**, developed with **Test-Driven Development (TDD)**.

## Features

- **Game Modes**
  - Player vs. Player
  - Player vs. Computer (3 difficulty levels: Easy, Normal, Hard)
- **Ship Placement**
  - Random placement (Random Spawn)
  - Manual ship placement
- **Turn-based Gameplay**
- **Tests with Jest**
